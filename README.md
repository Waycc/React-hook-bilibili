### 项目描述
这是使用react，react-hook，redux，saga开发的高仿bilibili，实现的功能：
1. 查看不同分类的视频列表
2. 搜索视频
3. 视频播放
4. 评论展示
5. 个别页面回退时缓存页面

### 安装
启动这个项目前，要先启动本项目的api服务：[react-bilibili-server](https://github.com/Waycc/react-bilibili-server)
安装依赖：
```bash
yarn
```
启动项目：

```bash
npm run start
```
